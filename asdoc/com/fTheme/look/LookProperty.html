Packagecom.fTheme.look
Classpublic class LookProperty
InheritanceLookProperty Object
Subclasses ColorArrayProperty, ColorProperty, FillProperty, NumberProperty, StrokeProperty

Represents one customization (style) property e.g. errorColor or applicationFill.

Child classes contain logic of parsing text value into strong-typed style value (e.g. errorColor into uint and applicationFill into IFill).

parseValue()method 
protected function parseValue(string:String):*

Parses text representation of style value into a strong-typed value e.g. parses "0xFFCC00 0x00CC00" into LinearGradient or "0xFFCC00" into uint (depends on child classes overrides).

By default the string is returned unchanged.
